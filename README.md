### Quick Start
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/vinay-arni20/DeepLearning-Specialization-Deeplearning.ai-Coursera.git
   cd DeepLearning-Specialization-Deeplearning.ai-Coursera
   ```

2. **Set Up Environment**:
   ```bash
   # Create virtual environment
   python -m venv dl_env
   source dl_env/bin/activate  # On Windows: dl_env\Scripts\activate

   # Install dependencies
   pip install numpy matplotlib tensorflow jupyter
   ```

3. **Explore the Content**:
   - **Assignments**: Navigate to course folders (e.g., `Course1-Neural-Networks-and-Deep-Learning/C1-Assignments/`)
   - **Notes**: Check the PDF files in each course directory for my detailed study notes
   - **Run Notebooks**: `jupyter notebook` and open any `.ipynb` file
